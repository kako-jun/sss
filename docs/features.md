# 主要機能

## 完全平等ランダム表示

すべての写真・動画を一巡するまで重複なしで表示します。一般的な「ランダム」は同じ写真が繰り返されがちですが、sss は全員に一度ずつ出番が来るまで次のラウンドを始めません。10万枚のコレクションでも、すべての写真が同じ頻度で表示されることを保証します。

## 差分スキャン

写真フォルダを登録すると、sss はファイルを自動検出します。初回だけフルスキャンを行い、2回目以降は前回から変わったファイルだけを処理します。10万枚規模でも通常5秒以内に起動でき、「重いから起動するのが億劫」という感覚を排除します。

## ファイル除外設定

スライドショーから除外したいフォルダやファイルをオーバーレイの「…」メニューから直接指定できます。「プライベートフォルダは見せたくない」「スクリーンショットは邪魔」「あの時期の写真は今は見たくない」といった細かい気持ちに応えます。除外ルールはアプリ内の SQLite データベースに保存されます。

撮影日付・ファイル単体・フォルダの3種類の除外方法から選べます。設定画面の「除外ルール」タブで登録済みルールの一覧表示・解除・手動追加もできます。

## お気に入り・SNS共有

スライドショー中に「これをピックしたい」と思ったとき、フッタのピックボタンからワンクリックで写真をコピーできます。コピー先はホームディレクトリの `Pictures/sss-picked` フォルダです。設定画面からコピー先のフォルダパスを変更することもできます。「…」メニューの「ピックを見る」からピックタブを直接開いて、ピック済み写真の一覧確認・削除もできます。

## 地図連携

GPS情報がある写真を表示しているとき、オーバーレイの地図セルに OpenStreetMap タイルを表示します。クリックで Google Maps を開けます。「どこで撮ったんだっけ」を確認できます。

## 統計グラフ

設定画面の統計タブで、全写真の表示回数を棒グラフで確認できます。完全平等ランダムが正しく動いていれば、写真間の表示回数のばらつきは極めて小さいはずです。「本当に公平か？」という疑問に、数値と視覚で答えます。

## 表示統計の記録

写真ごとに「何回表示されたか」「最後にいつ表示されたか」を記録します。「この写真、何度見たんだろう」という好奇心に答えるとともに、完全平等ランダムが正しく働いているかを数値で確認できます。

## スクリーンセーバー・スリープ抑制

アプリが起動している間、ディスプレイのスリープとスクリーンセーバーを自動的に抑制します。長時間の写真鑑賞や店舗サイネージで、気づいたら画面が消えていた、という状況を防ぎます。なお、ノートPCのフタを閉じた際のスリープは抑制しません（電力節約のため）。
