# 主要機能

## 完全平等ランダム表示

すべての写真・動画を一巡するまで重複なしで表示します。一般的な「ランダム」は同じ写真が繰り返されがちですが、sss は全員に一度ずつ出番が来るまで次のラウンドを始めません。10万枚のコレクションでも、すべての写真が同じ頻度で表示されることを保証します。

プレイリストの状態はアプリを終了しても保存され、次回起動時に続きから再開できます。

## 差分スキャン

写真フォルダを登録すると、sss はファイルを自動検出します。初回だけフルスキャンを行い、2回目以降は前回から変わったファイルだけを処理します。10万枚規模でも通常5秒以内に起動でき、「重いから起動するのが億劫」という感覚を排除します。

## ファイル除外設定（.sssignore）

ホームディレクトリに置いたテキストファイルで、スライドショーから除外したいフォルダやファイルを指定できます。「プライベートフォルダは見せたくない」「スクリーンショットは邪魔」「あの時期の写真は今は見たくない」といった細かい気持ちに応えます。

将来的には、見ている最中に画面から直接「このフォルダを除外」「この日付を除外」と操作できるようにする予定です。

## 表示統計の記録

写真ごとに「何回表示されたか」「最後にいつ表示されたか」を記録します。「この写真、何度見たんだろう」という好奇心に答えるとともに、完全平等ランダムが正しく働いているかを数値で確認できます。将来的には、グラフで一覧表示する機能も予定しています。

## スクリーンセーバー・スリープ抑制

アプリが起動している間、ディスプレイのスリープとスクリーンセーバーを自動的に抑制します。長時間の写真鑑賞や店舗サイネージで、気づいたら画面が消えていた、という状況を防ぎます。なお、ノートPCのフタを閉じた際のスリープは抑制しません（電力節約のため）。

## お気に入り・SNS共有（予定）

スライドショー中に「これをシェアしたい」と思ったとき、ボタンひとつで写真をピックアップできるようにします。選んだ写真はまとめて専用フォルダにコピーされ、SNS投稿にそのまま使えます。

## 地図連携（予定）

GPS情報がある写真を表示しているとき、オーバーレイ内に撮影場所の地図を直接表示します。「どこで撮ったんだっけ」を外部ブラウザを開かずに確認できます。
