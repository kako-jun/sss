# 対応環境

## 対応OS

| OS      | サポート状況                       |
| ------- | ---------------------------------- |
| Windows | 対応（Windows 10 以降を推奨）      |
| Linux   | 対応（主要ディストリビューション） |
| macOS   | 対応（macOS 11 以降を推奨）        |

Windows・Linux・macOSのすべてで同じ操作感を提供します。ファイルマネージャーとの連携もOS別に最適化されています。

## 対応ファイル形式

### 画像

JPEG、PNG、GIF、BMP、WEBP など主要な画像フォーマットに対応しています。EXIF情報（撮影日時・GPS座標・向き）を持つファイルは、その情報をオーバーレイに表示します。

### 動画

MP4、MOV、AVI、MKVなど主要な動画フォーマットに対応しています。動画はOSのネイティブ再生機能で表示されます。動画ファイルにはEXIF情報（撮影日時・GPS）は表示されず、ファイルとしての基本情報のみ表示されます。

## ディスプレイ

4Kディスプレイ（3840×2160）に最適化されています。それ以外の解像度でも動作します。4Kを超える写真は表示前に4Kサイズへリサイズしてキャッシュに保存することで、メモリ使用量を抑制します。

## ストレージ

ローカルディスク上のフォルダを対象とします。OSからマウントされたネットワークドライブも動作しますが、ネットワーク速度によっては表示が遅くなる場合があります。

## パフォーマンス目安

| 操作                       | 目安                      |
| -------------------------- | ------------------------- |
| 初回スキャン（10万枚）     | 30秒以内                  |
| 差分スキャン（変更100枚）  | 3秒以内                   |
| 起動からスライドショー開始 | 5秒以内（差分スキャン時） |
| 写真の切り替え             | 100ms以内                 |
| メモリ使用量               | 500MB以内（通常動作時）   |

## 制約事項

- **ファイル操作なし**: アプリからファイルを削除・移動・リネームする機能はありません。大切な写真コレクションを誤操作から守るための仕様です
- **マルチインスタンス不可**: 同一PC上で sss を複数同時起動することはできません
- **フタ閉じ時のスリープ**: スクリーンセーバーとディスプレイスリープは抑制しますが、ノートPCのフタを閉じた際のスリープは抑制しません（電力節約のため）
- **除外ルールの管理**: 除外ルールはアプリ内データベースに保存されます。フォルダごとの個別設定はできません

## 将来の対応プラットフォーム

**タブレット・モバイル**（予定）
Android タブレットおよびiPadへの対応を視野に入れています。デジタルフォトフレームとしてタブレットを活用するユースケースに応えます。スワイプで前後移動、タップでオーバーレイ表示など、タッチ操作への対応が必要です。
