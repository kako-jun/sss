# デザインガイドライン

新機能を実装する際にデザインの一貫性を保つために、現状のスタイル基調を定義します。

## 基本方針

- ダークテーマのみ（写真アプリなので写真を引き立てる）
- 「写真の邪魔をしない」UI。目立たせない
- 情報密度重視。コンパクトに収める

## カラー

| 用途                       | クラス            |
| -------------------------- | ----------------- |
| ベース背景                 | `bg-black`        |
| 主要テキスト               | `text-white/80`   |
| 通常テキスト               | `text-white/50`   |
| 補助テキスト               | `text-white/30`   |
| 極薄テキスト（アイコン等） | `text-white/25`   |
| ボーダー（標準）           | `border-white/8`  |
| ボーダー（薄め）           | `border-white/5`  |
| ボーダー（やや濃い）       | `border-white/10` |
| パネル背景（薄め）         | `bg-black/40`     |
| パネル背景（標準）         | `bg-black/50`     |
| パネル背景（濃い）         | `bg-black/80`     |

エラー表示も `text-red-400/80` 程度に抑える。カラフルなアクセントカラー（赤・青・緑等）は使わない。

## エフェクト

| 用途             | クラス                                   |
| ---------------- | ---------------------------------------- |
| ぼかし           | `backdrop-blur-md`（グラスモーフィズム） |
| ホバー時背景     | `hover:bg-black/70`                      |
| ホバー時テキスト | `hover:text-white/80`                    |
| アニメーション   | `transition-colors` `transition-opacity` |

## タイポグラフィ

| 用途               | クラス              |
| ------------------ | ------------------- |
| 数値・パス・時刻   | `font-mono`         |
| 標準フォントサイズ | `text-xs` `text-sm` |

## 形状

| 用途           | クラス                            |
| -------------- | --------------------------------- |
| 角丸           | `rounded`（小さめ、シャープ寄り） |
| アイコンサイズ | 13〜16px                          |

アイコンは Lucide React を使用。サイズ小さめ、`text-white/20`〜`/25` の薄めで使う。

## 禁止事項

- カラフルな色（赤・青・緑等のアクセントカラー）
- 太いボーダー・強い影
- 大きなボタン・派手なホバーエフェクト
- ライトテーマ
