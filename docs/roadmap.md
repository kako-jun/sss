# ロードマップ

## 近い将来に届けたいこと

### オーバーレイUIの常時表示

現在は「マウスを動かしたときだけ表示される」UIですが、これを画面下部に常時小さく表示するスタイルへ変えたいと考えています。撮影日時を大きく表示し、写真を眺めながらでも「いつ撮った写真か」が自然に目に入るようにします。マウスを動かすという「操作」をしなくても、情報がそこにある、という体験を目指します。

スライドショーの一時停止も変わります。現在はUIが出た瞬間に止まりますが、将来は「UIにマウスを乗せたとき」だけ止まるようにします。写真を眺めながらUIも見える、という自然な状態を作ります。

### UI上から直接除外できる

「このフォルダの写真は見たくない」と思ったとき、今は `.sssignore` ファイルを手動で編集する必要があります。これを、見ている画面から直接「このフォルダを除外」「この日付を除外」「このファイルだけ除外」と選べるようにします。感情が動いた瞬間にすぐ操作できることが大切です。

### お気に入りをSNSへ

スライドショーを見ていて「この写真、誰かに見せたい」と思ったとき、ワンクリックで写真をピックアップできるようにします。選んだ写真はまとめて専用フォルダにコピーされるので、あとでまとめてSNS投稿に使えます。写真を「発見する」喜びを、「伝える」喜びへつなげます。

### 地図をオーバーレイ内に埋め込む

GPS情報がある写真のとき、オーバーレイ内に撮影場所の地図を直接表示します。「どこで撮ったんだっけ」を確認するために外部ブラウザを開く必要がなくなります。写真と地図が並んで表示されることで、記憶がより鮮明に蘇る体験を目指します。

### 統計グラフで公平性を確認

全写真の表示回数をグラフで見られるようにします。完全平等ランダムが正しく動いていれば、写真間の表示回数のばらつきは極めて小さいはずです。「本当に公平か？」という疑問に、数値と視覚で答えます。

## もう少し先の展開

### タブレット・モバイル対応

デジタルフォトフレームとして部屋に置いたタブレットで sss を動かしたい、というユースケースに応えます。スワイプで前後移動、タップでオーバーレイ表示など、タッチ操作に最適化した体験を届けます。縦画面・横画面のどちらでも自然に使えるよう、画面サイズに柔軟に対応します。

### EXIF回転の選択

現在は写真に記録されたEXIF向き情報を自動的に反映しています。スキャナで取り込んだ画像など、EXIFの情報が実際の向きと異なるケースがあります。設定でこの自動回転のオン・オフを切り替えられるようにします。

### 動画のより深い対応

現在の動画表示は基本的な再生のみです。動画の長さに応じた表示時間の調整や、動画ファイルのメタ情報（長さ・解像度）のオーバーレイ表示など、写真と同じ水準の体験を動画にも届けます。
