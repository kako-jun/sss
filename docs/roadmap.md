# ロードマップ

## 近い将来に届けたいこと

### 除外ルール管理画面

現在はオーバーレイの除外ボタンで除外ルールを追加できますが、登録済みのルールを一覧表示したり解除したりする管理画面がまだありません。テキストリストで管理・解除できる画面を設定画面に追加します。

### 起動時の自動差分スキャン

現在は設定画面の「スキャン実行」ボタンを押すことでスキャンが始まります。アプリ起動時に前回のフォルダを自動的に差分スキャンし、ボタン操作なしでスライドショーを開始できるようにします。

### EXIF回転の設定

写真に記録されたEXIF向き情報の自動反映を、設定画面でオン・オフできるようにします。スキャナで取り込んだ画像など、EXIFの情報が実際の向きと異なるケースへの対応です。現状はオン固定です。

## もう少し先の展開

### タブレット・モバイル対応

デジタルフォトフレームとして部屋に置いたタブレットで sss を動かしたい、というユースケースに応えます。スワイプで前後移動、タップでオーバーレイ表示など、タッチ操作に最適化した体験を届けます。縦画面・横画面のどちらでも自然に使えるよう、画面サイズに柔軟に対応します。

### 動画のより深い対応

現在の動画表示は基本的な再生のみです。動画の長さに応じた表示時間の調整や、動画ファイルのメタ情報（長さ・解像度）のオーバーレイ表示など、写真と同じ水準の体験を動画にも届けます。
