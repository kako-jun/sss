# 使い方ガイド

## 初回起動

1. アプリを起動すると設定画面が表示されます
2. 「フォルダ選択」ボタンで写真・動画が入ったフォルダを選びます
3. 「フォルダをスキャン」ボタンでスキャンを開始します
4. スキャン完了後、自動的にスライドショーが始まります

## 画面構成

### スライドショー画面

起動後、写真・動画が最大化ウィンドウで表示されます。背景は黒、写真はアスペクト比を保ったまま中央に配置されます。

画面下部には常時オーバーレイUIが表示されています。マウスカーソルをオーバーレイに乗せるとスライドショーが一時停止します。オーバーレイから離れると再開します。

### オーバーレイUI

画面下部に常時表示される半透明のUIです。写真を邪魔せず、ファイル情報と操作ボタンを提供します。

オーバーレイにマウスを乗せている間はスライドショーが自動的に一時停止します。離れると再開します。

**表示される情報**

| 項目 | 内容 |
|---|---|
| ファイルパス | ストレージ上のフルパス |
| 画像サイズ | 幅 × 高さ（ピクセル） |
| ファイルサイズ | ファイルの容量 |
| 撮影日時 | EXIFから取得（写真のみ） |
| GPS座標 | 撮影場所の緯度・経度（写真のみ、情報がある場合） |
| プレイリスト位置 | 現在の位置 / 総数（例: 1,234 / 100,000） |
| 表示回数 | これまでに表示された回数 |
| 最終表示日時 | 前回表示された日時 |

動画ファイルの場合、撮影日時・GPS情報は表示されません。

**操作ボタン**

| ボタン | 動作 |
|---|---|
| 前へ | 履歴をさかのぼって前の写真に戻る（表示回数はカウントしない） |
| 次へ | 即座に次の写真へ進む |
| 開く | OSのファイルマネージャーで該当ファイルを選択状態で開く |
| シェア | ファイルを `~/Pictures/sss` フォルダにコピーする |
| 除外 | 現在の写真をスライドショーから除外する（3種類の除外方法を選択） |
| 設定 | 設定画面を開く |

**除外メニュー**

「除外」ボタンを押すと3種類の除外方法から選べます：

| 選択肢 | 動作 |
|---|---|
| この日付を除外 | EXIF撮影日付で除外（`**/YYYY-MM-DD/**` パターン） |
| このファイルを除外 | ファイル名パターンで除外 |
| このフォルダを除外 | 親フォルダパスで除外 |

いずれの方法も `.sssignore` ファイルにパターンが追記されます。

### 設定画面

| 項目 | 説明 |
|---|---|
| フォルダ選択 | スライドショー対象のフォルダをダイアログで選択 |
| スキャン実行 | 選択したフォルダをスキャンしてファイルを検出 |
| スキャン結果 | 追加・更新・削除されたファイル数と総ファイル数を表示 |
| 表示間隔 | スライドショーの切り替え間隔（5〜60秒） |
| シェア先フォルダ | シェアボタンでコピーされる先のフォルダパス（デフォルト: `~/Pictures/sss`） |
| 統計グラフ | 全写真の表示回数を棒グラフで表示 |

## キーボードショートカット

| キー | 動作 |
|---|---|
| 左矢印 | 前の写真へ戻る |
| 右矢印 | 次の写真へ進む |
| ESC | アプリを終了 |

## .sssignore による除外設定

ホームディレクトリに `.sssignore` という名前のテキストファイルを置くことで、特定のフォルダやファイルをスライドショーから除外できます。書き方は `.gitignore` と同じルールです。

```
# 特定の日付を除外
**/2023-05-15/**

# プライベートフォルダを除外
**/private/**

# スクリーンショットを除外
screenshot_*.png
```

オーバーレイの「除外」ボタンからも同じファイルにパターンが追記されます。

## サイネージ用途での運用

店舗や施設での常時表示など、自動起動・終了が必要な場合は OSの機能を利用します。アプリ内部にタイマー機能は持っていません。

- **Windows**: タスクスケジューラーで起動・終了を設定
- **Linux**: cron または systemd timer で管理
- **macOS**: launchd の plistファイルで設定
