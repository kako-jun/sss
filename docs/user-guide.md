# 使い方ガイド

## 初回起動

1. アプリを起動すると設定画面が表示されます
2. 「フォルダ選択」ボタンで写真・動画が入ったフォルダを選びます
3. 「フォルダをスキャン」ボタンでスキャンを開始します
4. スキャン完了後、自動的にスライドショーが始まります

## 画面構成

### スライドショー画面

起動後、写真・動画がフルスクリーンで表示されます。背景は黒、写真はアスペクト比を保ったまま中央に配置されます。

オーバーレイUIはデフォルト非表示です。マウスを動かすと画面下部にフェードインし、3秒間マウスが動かないとフェードアウトします。設定ボタン（⚙️）は右上に常時表示されています。

### オーバーレイUI

画面下部にフェードイン/アウトする半透明のバーです。4列×2行のグリッドで情報と操作を提供します。

オーバーレイにマウスを乗せている間はフェードアウトせず、スライドショーも自動的に一時停止します。離れると3秒タイマーが再開し、スライドショーも再開します。

**上行（情報）**

| セル       | 内容                                                                 |
| ---------- | -------------------------------------------------------------------- |
| 地図       | GPS情報がある場合は OpenStreetMap タイル表示。ない場合はグレーアウト |
| 撮影日時   | EXIFから取得（デジタル時計風の大きめ表示）                           |
| ファイル名 | ファイル名とファイルサイズ                                           |
| 位置/回数  | プレイリスト位置 / 総数、表示回数                                    |

**下行（操作）**

| ボタン | 動作                                  |
| ------ | ------------------------------------- |
| …      | サブメニューを開く（下記参照）        |
| ⏸/▶    | スライドショーの一時停止/再開をトグル |
| 前へ   | 履歴をさかのぼって前の写真に戻る      |
| 次へ   | 即座に次の写真へ進む                  |

**… サブメニュー**

| 項目                       | 動作                                                   |
| -------------------------- | ------------------------------------------------------ |
| ファイルマネージャーで開く | OSのファイルマネージャーで該当ファイルを選択状態で開く |
| ピック（コピー）           | ファイルを `~/Pictures/sss-picked` フォルダにコピー    |
| ウィンドウモード切り替え   | フルスクリーンとウィンドウモードを切り替える           |
| 設定                       | 設定画面を開く                                         |
| 撮影日付で除外             | EXIF撮影日付で除外（`**/YYYY-MM-DD/**` パターン）      |
| このフォルダを除外         | 親フォルダパスで除外                                   |
| このファイルを除外         | ファイル名パターンで除外                               |

除外ルールはデータベースに追記されます。

### 設定画面

| 項目             | 説明                                                                       |
| ---------------- | -------------------------------------------------------------------------- |
| フォルダ選択     | スライドショー対象のフォルダをダイアログで選択                             |
| スキャン実行     | 選択したフォルダをスキャンしてファイルを検出                               |
| スキャン結果     | 追加・更新・削除されたファイル数と総ファイル数を表示                       |
| 表示間隔         | スライドショーの切り替え間隔（5〜60秒）                                    |
| シェア先フォルダ | シェアボタンでコピーされる先のフォルダパス（デフォルト: `~/Pictures/sss`） |
| 統計グラフ       | 全写真の表示回数を棒グラフで表示                                           |

## キーボードショートカット

| キー   | 動作           |
| ------ | -------------- |
| 左矢印 | 前の写真へ戻る |
| 右矢印 | 次の写真へ進む |
| ESC    | アプリを終了   |

## 除外設定

除外ルールはアプリ内の SQLite データベースに保存されます。オーバーレイの「…」メニューから直接パターンを追加できます。

以前のバージョンで `~/.sssignore` を使用していた場合、初回スキャン時に自動的にデータベースへ移行されます（元ファイルは `~/.sssignore.bak` にリネームされます）。

## サイネージ用途での運用

店舗や施設での常時表示など、自動起動・終了が必要な場合は OSの機能を利用します。アプリ内部にタイマー機能は持っていません。

- **Windows**: タスクスケジューラーで起動・終了を設定
- **Linux**: cron または systemd timer で管理
- **macOS**: launchd の plistファイルで設定
