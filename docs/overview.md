# sss — Smart Slide Show

## アプリの概要

**sss（Smart Slide Show）** は、10万枚以上の写真・動画コレクションを「公平に」楽しむためのデスクトップ向けスライドショーアプリです。

## 解決したい課題

長年写真を撮り続けていると、気づけば10万枚を超えるコレクションが手元に積み上がる。でも、ふだんのスライドショーアプリでは「ランダム表示」と言いながら、同じ写真ばかり出て、何年撮り続けてきた思い出が全然表示されないまま埋もれていく。

どの写真にも平等に出番を与えたい。あの日撮った一枚も、ずっと忘れていた旅の写真も、10年後に見返したとき「こんな写真あったんだ」と発見できる喜びを、すべての写真に届けたい。それが sss を作った理由です。

## ビジョン

- **一枚も取りこぼさない**。10万枚のコレクションでも、すべての写真が均等に表示される
- **すぐ始められる**。2回目以降の起動は5秒以内。思い立ったときにサッと写真の世界に入れる
- **鑑賞に集中できる**。余計なUIは消えていて、写真だけが全画面に広がる
- **ふと知りたくなる瞬間**に答える。「これいつ撮ったんだろう」「どこだっけ」をマウスひとつで確認できる
- **ずっと表示し続けられる**。店舗やリビングでのサイネージ用途でも、スリープせず写真を届け続ける

## 対象ユーザー

- 何万枚もの写真を持ち、すべての思い出を平等に振り返りたい個人
- 店舗・施設でデジタルサイネージとして写真を常時表示したい人
- 「どうせランダムは偏る」と諦めていた写真コレクターへ
